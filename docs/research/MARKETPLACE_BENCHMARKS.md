# Marketplace Benchmarks — Reference Index

Curated set of marketplace / app-store references for the **A360 Agent Exchange** concept.
Saved 2026-07-09. Companion review findings live in `MARKETPLACE_REVIEW.md`.

> These are UX / information-architecture / trust-pattern benchmarks. We are building a
> marketplace of **AI agents for medical aesthetics practices that layer into the EMR** — so
> the healthcare/EMR marketplaces (athenahealth, Epic) and the best-in-class app stores
> (Shopify, Salesforce) are the highest-signal references.

## Horizontal SaaS / CRM marketplaces

| Marketplace | URL | Why it matters to us |
|---|---|---|
| Salesforce AppExchange | https://appexchange.salesforce.com/ | Gold standard for enterprise app detail pages, trust badges, reviews |
| HubSpot Marketplace | https://ecosystem.hubspot.com/marketplace | Clean category IA, "built by" attribution, vertical SaaS feel |
| Atlassian Marketplace | https://marketplace.atlassian.com/ | Developer-partner model, pricing tiers, cloud-fortified security badges |
| Monday.com Marketplace | https://monday.com/marketplace | Visual card grid, use-case-led browsing |
| Zendesk Marketplace | https://www.zendesk.com/marketplace/ | Category + role filtering, integration-first framing |
