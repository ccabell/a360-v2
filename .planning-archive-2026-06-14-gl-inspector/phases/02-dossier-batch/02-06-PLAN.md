    **STRUCTURED_COVERAGE.md:**

    Run the verification SQL from STRUCTURED_EMISSION_ADDENDUM.md against Supabase project `aejskvmpembryunnbgrk`:

    ```sql
    SELECT p.name,
      (SELECT COUNT(*) FROM item_concerns ic WHERE ic.offering_id=p.id) AS concerns,
      (SELECT COUNT(*) FROM item_concerns ic WHERE ic.offering_id=p.id AND ic.is_fda_indicated=true) AS fda_indicated,
      (SELECT COUNT(*) FROM item_body_areas iba WHERE iba.offering_id=p.id) AS body_areas,
      (SELECT COUNT(*) FROM item_body_areas iba WHERE iba.offering_id=p.id AND iba.anatomy_specificity='precise') AS zone_level,
      COALESCE(array_length(p.does_not_solve,1),0) AS limitations,
      (SELECT COUNT(*) FROM agent_reference_docs ard WHERE ard.offering_id=p.id AND ard.status='draft') AS docs
    FROM products p
    WHERE p.kind='product'
    ORDER BY p.name;
    ```

    Also count new aliases per product:
    ```sql
    SELECT p.name,
      (SELECT COUNT(*) FROM aliases a
       JOIN item_concerns ic ON a.concern_id=ic.concern_id
       WHERE ic.offering_id=p.id) AS concern_aliases,
      (SELECT COUNT(*) FROM aliases a
       JOIN item_body_areas iba ON a.body_area_id=iba.body_area_id
       WHERE iba.offering_id=p.id) AS body_area_aliases
    FROM products p WHERE p.kind='product' ORDER BY p.name;
    ```

    Also run the FDA-indicated validation:
    ```sql
    SELECT COUNT(*) FROM item_concerns ic
    WHERE ic.is_fda_indicated = true
    AND NOT EXISTS (SELECT 1 FROM evidence_links el
                    WHERE el.offering_id = ic.offering_id AND el.source='fda_label');
    -- MUST be 0
    ```

    Write STRUCTURED_COVERAGE.md with:
    - Per-product table: name, # item_concerns (FDA-indicated count), # item_body_areas (by specificity), # new aliases, does_not_solve count, # docs
    - FLAGGED products: any with <3 concerns or zero zone-level anatomy
    - FDA validation result
    - Summary totals

    **TAXONOMY_ADDITIONS.md:**

    Query for all additions made during the batch:
    ```sql
    -- New concerns added (check created_at > batch start date)
    SELECT name, category, patient_description, created_at FROM concerns
    WHERE created_at > '2026-06-12T00:00:00Z'
    ORDER BY created_at;

    -- New body_areas added
    SELECT ba.name, ba.level, parent.name as parent_name, ba.created_at FROM body_areas ba
    LEFT JOIN body_areas parent ON parent.id = ba.parent_id
    WHERE ba.created_at > '2026-06-12T00:00:00Z'
    ORDER BY ba.created_at;

    -- New aliases added
    SELECT a.phrase, a.normalized, c.name as concern_name, ba.name as body_area_name, a.created_at
    FROM aliases a
    LEFT JOIN concerns c ON c.id = a.concern_id
    LEFT JOIN body_areas ba ON ba.id = a.body_area_id
    WHERE a.created_at > '2026-06-12T00:00:00Z'
    ORDER BY a.created_at;
    ```

    Write TAXONOMY_ADDITIONS.md with:
    - New concerns: name, type, category, patient_description, which product triggered it
    - New body_areas: name, level, parent, which product triggered it
    - New aliases: phrase, canonical target (concern or body_area), which product triggered it
    - Summary counts: N new concerns, N new body_areas, N new aliases

    1. Query source_registry for all sources logged during the batch:
    ```sql
    SELECT name, publisher, source_kind, authority_rank, rights_class, status, added_by
    FROM source_registry
    ORDER BY authority_rank, name;
    ```

    2. Query ingestion_queue for all queued items:
    ```sql
    SELECT title, doi, url, rights_class, discovered_during, status
    FROM ingestion_queue
    ORDER BY status, title;
    ```

    3. Count by authority_rank and source_kind:
    ```sql
    SELECT authority_rank, source_kind, COUNT(*) as count
    FROM source_registry
    WHERE added_by = 'discovery'
    GROUP BY authority_rank, source_kind
    ORDER BY authority_rank, source_kind;
    ```

    4. **TOP 10 HIGHEST-AUTHORITY OA PAPERS (CRITICAL USER REQUIREMENT):**

    Query for OA papers from the priority journals (JCAD, Cureus, Dermatology and Therapy, ASJ) that are relevant to Botox/Neurotoxins:
    ```sql
    SELECT iq.title, iq.doi, iq.url, iq.rights_class, sr.authority_rank, sr.name as journal
    FROM ingestion_queue iq
    LEFT JOIN source_registry sr ON sr.name ILIKE '%' || split_part(iq.discovered_during, '_', 1) || '%'
    WHERE iq.rights_class IN ('open_access_cc_by', 'public_domain')
    ORDER BY sr.authority_rank ASC, iq.title
    LIMIT 20;
    ```

    AND el.doi IS NOT NULL
    ORDER BY el.authority_rank;
