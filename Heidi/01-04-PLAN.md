## OBJECTIVE

The distribution story: `/embed/ask` — a chrome-less variant designed to live inside any allowed host's iframe, speaking a minimal origin-checked postMessage protocol — plus an internal Boulevard-styled host page proving the pitch ("this drops into boulevard.com"), and the demo dry-run checklist that closes Phase 01.

This mirrors the exact pattern reverse-engineered from Heidi: their public page is an iframe of `scribe.heidihealth.com/chat/embed?query=…` with origin-allowlisted `heidi:*` messages. Ours is original code implementing the same architecture.

### T2 — Bridge
`lib/embed/bridge.ts`:

```ts
const OUT = ['a360:ready','a360:ask_sent','a360:answer_complete','a360:citation_click','a360:resize'] as const

export function createBridge() {
  const allowed = (process.env.NEXT_PUBLIC_EMBED_PARENT_ORIGINS ?? '')
    .split(',').map(s => s.trim()).filter(Boolean)
  const inIframe = typeof window !== 'undefined' && window.self !== window.top
  const post = (type: typeof OUT[number], payload?: unknown) => {
    if (!inIframe) return
    // targetOrigin '*' is acceptable for these non-sensitive notifications,
    // but prefer the first allowed origin when configured:
    window.parent.postMessage({ type, payload }, allowed[0] ?? '*')
  }
  const onSeed = (handler: (q: string) => void) => {
    window.addEventListener('message', (e) => {
      if (!allowed.includes(e.origin)) { console.warn('[a360-embed] rejected origin', e.origin); return }
      if (e.data?.type === 'a360:seed' && typeof e.data.query === 'string') handler(e.data.query)
    })
  }
  return { post, onSeed }
}
```

Wire in the embed page: `post('a360:ready')` on mount; `onAskSent` → `a360:ask_sent`; `onAnswerComplete` → `a360:answer_complete`; `onCitationClick` → `a360:citation_click`; a `ResizeObserver` on the content root → throttled `a360:resize { height }`. (Hooks already exist on `<AskExperience>` from 01-01.)

### T3 — Frame-ancestors CSP

Confirm no global `X-Frame-Options: DENY` overrides it for this path. All non-embed routes keep their current framing protections.
