The `handleSave` function:
```typescript
const handleSave = async (updates: Partial<{ content: Record<string, any>; review_status: string }>) => {
  setSaving(true)
  try {
    const res = await fetch(`/api/fuel-docs/${id}`, {
      method: "PATCH",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify(updates),
    })
    if (!res.ok) throw new Error("Failed to save")
    await fetchDoc()  // refetch
  } catch (err) {
    // show error
  } finally {
    setSaving(false)
  }
}
```

For tab save flow: each tab component manages its own local state for edited values. On "Save" within the tab, it calls `onSave({ content: { ...doc.content, ...editedFields } })` to merge the edited fields into the existing content JSONB.
