**Step 1 — Create `components/grounding/markdown-table.tsx`**

Create a new component that parses a raw GFM pipe table string and renders it as an HTML `<table>` with per-cell citation badges.

Props interface:
```typescript
interface MarkdownTableProps {
  raw: string;
  resolve: (id: string) => ResolvedRef | null;
  byNumber: Map<number, ResearchCitation>;
  citations: ResearchCitation[];
}
```

Implementation:
- `parseRow(line: string): string[]` — strips leading/trailing `|`, splits on `|`, trims each cell
- Split `raw` by `\n` into lines. Line 0 = header, line 1 = separator (skip), lines 2+ = data rows
- Filter out any empty lines (LLM sometimes adds blank lines inside tables)
- Each cell rendered via `parseCitationSegments(cellText, resolve, byNumber, citations)` from `./citation-segments`
- Wrap entire `<table>` in `<div className="overflow-x-auto my-3">` for mobile scroll (ANS-04)
- Table classes: `min-w-full border-collapse text-sm`
- Header row: `<tr className="border-b border-border bg-muted/50">`, each `<th>` gets `px-3 py-2 text-left font-semibold text-foreground`
- Data rows: `<tr className="border-b border-border last:border-0 odd:bg-background even:bg-muted/20">`, each `<td>` gets `px-3 py-2 align-top text-foreground/90`

Export the component as a named export: `export function MarkdownTable(...)`.

// Table detection: requires header row + separator row (no false positives)
const isTable = /^\|.+\|\n\|[-| :]+\|/m.test(trimmed);

    <MarkdownTable
      key={pi}
      raw={trimmed}
      resolve={resolve}
      byNumber={byNumber}
      citations={citations ?? []}
    />
  );
}
